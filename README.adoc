== About

Configures, builds, and runs an external server.
The server runs Transmission, OpenVPN, Nginx.

== Install

[,sh]
----
git clone git@github.com:rdavid/argus.git &&
  cd ./argus &&
  cat <<EOT >> cred.env
USER=[username]
PASS=[password]
URL=[url]
EOT
sudo docker-compose up -d
----
